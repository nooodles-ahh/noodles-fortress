# Noodles' Fortress
This is my personal fork of Pre-Fortress 2 based on my copy of the code dated April 6th, 2022. I feel it has been long enough now where I feel comfortable releasing this without impacting the current project, in spite of how little the game has noticeably changed since I left.

The intention with this is to finish everything I wanted to finish, implementing anything else I feel like and fixing all the bugs I can be bothered with. This is being tested against the 0.7.3 build of the game, the download can be found on the official site [here](https://dl.prefortress.com/0.7/pf2_0.7.3.7z). The majority of game assets were not created by me so it is not appropriate for me to redistribute them without permission.

[This fork is officially not endorsed by the developers](https://github.com/nooodles-ahh/noodles-pre-fortress/issues/1), but is endorsed by me, as I did a very significant amount of the code. The original intention for 0.7 was to make it open source just as 0.6 was, but "making Pre-Fortress 2 version 0.6 open source was a mistake" apparently. The code is based on a leaked 2008 version of the TF2 source code, I don't think this should really be an issue as almost all functionality is present in the public SDK, but is something to consider.

My ideas and progress on the mod can be tracked under my project [here](https://github.com/users/nooodles-ahh/projects/3).
